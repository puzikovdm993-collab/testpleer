# Аудио Плеер

Современный веб-аудиоплеер с поддержкой плейлистов, перемешивания и повторения треков.

## 🎵 Функции

- **Воспроизведение аудио**: Play/Pause, переключение треков
- **Плейлист**: Добавление, удаление, отображение треков
- **Прогресс-бар**: Отображение и перемотка трека
- **Громкость**: Регулировка + mute/unmute (клавиша `M`)
- **Перемешивание (Shuffle)**: Алгоритм Fisher-Yates
- **Повтор (Repeat)**: 3 режима: выкл/все/один
- **Горячие клавиши**: 
  - `Пробел` — Play/Pause
  - `←` / `→` — Предыдущий/Следующий трек
  - `↑` / `↓` — Увеличить/Уменьшить громкость
  - `M` — Mute/Unmute
- **LocalStorage**: Сохранение настроек и метаданных
- **Экспорт/Импорт**: JSON-файлы плейлиста
- **Адаптивность**: Mobile-friendly дизайн

## 🚀 Быстрый старт

1. Откройте `index.html` в браузере
2. Загрузите аудиофайлы через кнопку "📁 Добавить треки"
3. Наслаждайтесь музыкой!

## 🔧 Технические детали

### Исправленные проблемы

1. **Утечка памяти**: Добавлена очистка blob URL через `URL.revokeObjectURL()` при удалении треков
2. **Логика shuffle**: Исправлена обработка случая, когда текущий трек не найден в shuffledIndices
3. **Магические числа**: Все числовые значения вынесены в константы (`VOLUME_STEP`, `DEFAULT_VOLUME`, `MAX_VOLUME`, `RESTART_THRESHOLD`)
4. **Обработка ошибок**: Добавлены try-catch блоки и обработка промисов для `audioPlayer.play()`
5. **Состояние громкости**: Добавлено централизованное управление состоянием громкости и mute
